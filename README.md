# Javascript-Projeleri

HTML, CSS ve JavaScript kullanılarak hazırlanmış, başlangıç ve orta seviye için uygun 32 adet örnek proje koleksiyonu. Bu projeler DOM manipülasyonu, API kullanımı, CSS animasyonları ve temel algoritma mantığını kavramak için hazırlanmıştır.

## Proje Listesi

### Temel CSS ve JS Efektleri
1. **Arkaplan Rengi Değiştirme**
   - Butona basıldığında sayfanın arka plan rengini rastgele değiştiren uygulama.
   
2. **Karanlık Mod (Dark Mode)**
   - CSS değişkenleri kullanılarak tasarlanmış, tema değiştirme (aydınlık/karanlık) özelliği.
   
3. **Açılır Menü**
   - Responsive tasarımlar için mobil uyumlu, animasyonlu menü çubuğu.
   
4. **Resim Yakınlaştırma (Zoom Effect)**
   - E-ticaret sitelerinde sıkça kullanılan, mouse ile resmin üzerine gelindiğinde yakınlaştırma efekti.
   
5. **Genişleyen Kartlar**
   - Tıklandığında akordiyon şeklinde açılan ve diğerlerini küçülten şık resim kartları.
   
6. **Buton Animasyonu**
   - Buton üzerine gelindiğinde veya tıklandığında oluşan CSS ripple (dalga) efekti.
   
7. **Buton Efekti**
   - Farklı etkileşimlere sahip yaratıcı buton tasarımları.

8. **İlerleme Adımları (Progress Steps)**
   - Çok aşamalı formlarda kullanılan, "İleri" ve "Geri" butonlu ilerleme çubuğu.

### Metin ve Bildirim İşlemleri
9. **Otomatik Yazı Efekti**
   - Sayfa açılışlarında veya sloganlarda kullanılan, daktilo gibi harf harf yazılan metin animasyonu.
   
10. **Mesaj Bilgisi (Toast Notification)**
    - "Sepete eklendi" gibi işlemler için ekranın köşesinde belirip kaybolan bildirim kutusu.
    
11. **Modal Mesajı**
    - Butona tıklandığında arkaplanı flu yapıp öne çıkan açılır pencere (popup).
    
12. **Bilgi Kutusu**
    - Kullanıcıya ipucu veya detay bilgi veren hover etkileşimli kutular.

### Görsel ve Medya Uygulamaları
13. **Kalp Akışı**
    - Sosyal medya beğenilerini simüle eden, çift tıklandığında veya butona basıldığında ekranda uçuşan kalp animasyonu.
    
14. **Resim Geçişi (Slider)**
    - Resimler arasında otomatik veya manuel geçiş yapılabilen slide yapısı.
    
15. **Oto Resim Geçişi**
    - Belirli aralıklarla kendiliğinden değişen arka plan veya görsel slayt gösterisi.
    
16. **Ses Efekti / Ses Değiştirme Butonu**
    - Tıklandığında belirli sesleri çalan (alkış, yuhalama vb.) interaktif butonlar (Sound Board).

### API ve Veri Tabanlı Uygulamalar
17. **Film Uygulaması**
    - Film API'si (TMDB vb.) kullanılarak film afişlerini, puanlarını ve özetlerini listeleyen uygulama.
    
18. **Github Profil Uygulaması**
    - GitHub API kullanarak, kullanıcı adı girilen kişinin profil bilgilerini ve depolarını getiren arama motoru.
    
19. **Github Profil Görüntüleyici**
    - Profil kartı tasarımına odaklanmış alternatif bir görüntüleyici.
    
20. **Hava Durumu Uygulaması**
    - Girilen şehrin anlık hava durumu bilgisini API üzerinden çeken uygulama.
    
21. **Yemek Tarifi Uygulaması**
    - Yemek API'si kullanarak rastgele veya aranan yemeğin tarifini ve malzemelerini gösteren uygulama.
    
22. **Şaka Matik (Dad Jokes)**
    - API'den rastgele soğuk şakalar çeken ve butona basıldığında yenisini getiren eğlenceli uygulama.

### Araçlar ve Yardımcı Uygulamalar
23. **Not Uygulaması**
    - Tarayıcı hafızasına (LocalStorage) kayıt yapabilen, ekle/sil/düzenle özellikli not defteri.
    
24. **Not Defteri**
    - Alternatif arayüze sahip, hızlı not alma aracı.
    
25. **Yapılacaklar Listesi (Todo List)**
    - (Klasörlerde 'Metin' veya benzeri bir isimle olabilir) Görev ekleme, çizme ve silme uygulaması.
    
26. **Şifre Üretme Uygulaması**
    - Büyük harf, küçük harf, sayı ve sembol seçenekleriyle güvenli şifre oluşturan araç.
    
27. **Geri Sayım Saati**
    - Belirlenen bir tarihe veya yeni yıla ne kadar kaldığını gösteren sayaç.
    
28. **Sayı İle Yükleme Ekranı**
    - Sayfa yüklenirken 0'dan 100'e kadar sayan ve arkaplanı netleştiren yükleme animasyonu.
    
29. **Yükleme Ekranı (Loading)**
    - İçerik gelmeden önce gösterilen iskelet (skeleton) veya dönen yükleme ikonları.

30. **Çizim Uygulaması**
    - HTML5 Canvas kullanılarak mouse ile ekranda çizim yapmayı sağlayan uygulama.
    
31. **Soru Uygulaması (Quiz App)**
    - Çoktan seçmeli soruları içeren, doğru/yanlış sayısını tutan ve skor gösteren yarışma.
    
32. **Şans Çarkı**
    - Rastgele seçim yapmak için dönen çark animasyonu.
